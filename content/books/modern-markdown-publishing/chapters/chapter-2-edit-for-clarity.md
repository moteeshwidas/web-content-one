# Chapter 2: Edit for Clarity

Editing is where good drafts become useful resources.

## Practical Editing Rules

1. One paragraph, one idea.
2. Prefer active voice.
3. Keep examples near the concept they explain.
4. Cut any sentence that does not help the reader act.

Repeat this pass twice, once for logic and once for language.
